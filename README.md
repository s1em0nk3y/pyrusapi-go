Library to work with Pyrus API v4 written in Golang.

## Install
`go get github.com/s1em0nk3y/pyrusapi-go`

## Quick Start

```go
package main

import (
	"fmt"
	"log"

	"github.com/s1em0nk3y/pyrusapi-go"
)

func main() {
	c, err := pyrus.NewClient("bot_login", "bot_security_key")
	if err != nil {
		log.Fatalln(err)
	}

	p, err := c.Profile()
	if err != nil {
		log.Fatalln(err)
	}

	fmt.Println(p.FirstName)
}
```
